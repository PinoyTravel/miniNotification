h1. miniNotification jQuery Plugin (written in CoffeeScript)

miniNotification is very basic but powerful notification jQuery plugin that allows you to fadeIn or slide a notification message at any time.

With miniNotification, you'll be able to display successful or error messages to users in a nice and friendly way (see Demo)
