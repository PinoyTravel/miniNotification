h1. miniNotification jQuery Plugin (written in CoffeeScript)

miniNotification is very basic but powerful notification jQuery plugin that allows you to fadeIn or slide a notification message at any time.

With miniNotification, you'll be able to display successful or error messages to users in a nice and friendly way (see Demo)

If you have any question of find a bug, feel <a href="https://github.com/MatthieuA/Mini-Notification/issues">hit me up</a> at GitHub or <a href="http://www.twitter.com/matthieu_tweets">shoot me a tweet</a>..

miniNotification is compatible with all modern web browsers including; Internet Explorer 6/7/8/9, Firefox 3+, Chrome, Safari and Mobile Safari.
